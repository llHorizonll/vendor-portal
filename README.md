## Installation

before use pnpm if you doesn't have pnpm install if first https://pnpm.io/installation

if you have npm

```
 npm install -g pnpm
```

**Installation using pnpm:**

```
 pnpm install
```

## How to use

**Start on Local:**

```
 pnpm dev
```

**Build Project:**

```
 pnpm build
```

**Serve in Build Project:**

```
 pnpm preview
```
